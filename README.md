# Odin Landing Page

A landing page built from scratch using HTML and CSS as part of The Odin Project Foundations curriculum.

[View on GitHub Pages](#) ← add your link here

## Features

- Fully custom layout built with Flexbox
- Responsive header with logo and navigation links
- Hero section with headline, subtext, and call to action button
- Information section with image cards
- Quote/testimonial section
- Call to action section
- Footer

## Technologies

- HTML
- CSS (Flexbox)

## What I Learned

- Building complete page layouts with Flexbox
- Nesting flex containers inside each other
- Working with colors, typography, and spacing from a design spec
- Translating a visual design into code
- CSS specificity and the cascade
